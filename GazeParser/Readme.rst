Please read `<http://gazeparser.sourceforge.net/ins/index.html>`_ for detail.

For Windows users
==================

* Install either of these files.
    - GazeParser-0.8.2/GazeParser-0.8.2-win32.exe
    - GazeParser-0.8.2/GazeParser-0.8.2.zip

* If you plan to record gaze data, install one of following files that is suitable for your camera.
    - SimpleGazeTracker-0.8.2/SimpleGazeTracker-0.8.2-OptiTrack-1.2.msi
    - SimpleGazeTracker-0.8.2/SimpleGazeTracker-0.8.2-OpenCV-2.4.3.msi
    - SimpleGazeTracker-0.8.2/SimpleGazeTracker-0.8.2-GPC5300-2.21-13.msi
    - SimpleGazeTracker-0.8.2/SimpleGazeTracker-0.8.2-FlyCapture2-2.5.4.msi

For Linux users
============================

* Install following file.
    - GazeParser-0.8.2/GazeParser-0.8.2.zip

* If you plan to record gaze data, download source code for SimpleGazeTracker and build it.
    - SimpleGazeTracker-0.8.2/SimpleGazeTracker-0.8.2.tar.gz

.. note::
    Debian .deb packages of GazeParser and SimpleGazeTracker are testing now.
    Please try them (built on Ubuntu 14.04LTS).
    
    - GazeParser-0.8.2/gazeparser_0.8.2_all.deb
    - SimpleGazeTracker-0.8.2/simplegazetracker_0.8.2_amd64.deb
