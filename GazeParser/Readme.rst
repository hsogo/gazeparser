Please read `<http://gazeparser.sourceforge.net/ins/index.html>`_ for detail.

For Windows users
==================

* Install either of these files.
    - GazeParser-0.6.3/GazeParser-0.6.3-win32.exe
    - GazeParser-0.6.3/GazeParser-0.6.3.zip

* If you plan to record gaze data, install one of following files that is suitable for your camera.
    - SimpleGazeTracker-0.6.3/SimpleGazeTracker-0.6.3-optitrack-1.3.038.msi
    - SimpleGazeTracker-0.6.3/SimpleGazeTracker-0.6.3-opencv-2.4.3.msi
    - SimpleGazeTracker-0.6.3/SimpleGazeTracker-0.6.3-GPC5300-1.70-08.msi
    - SimpleGazeTracker-0.6.3/SimpleGazeTracker-0.6.3-FlyCapture2-2.3.19.msi

For Linux and MacOS X users
============================

* Install following file.
    - GazeParser-0.6.3/GazeParser-0.6.3.zip

* If you plan to record gaze data, download source code for SimpleGazeTracker and build it.
    - SimpleGazeTracker-0.6.3/SimpleGazeTracker-0.6.3.tar.gz
